## 📌 About The Project

A modern implementation of the classic **Tic Tac Toe (XO)** game using **C# Windows Forms**.
This project focuses on clean code structure, game logic handling, and user-friendly UI.

---

## ✨ Features

✔️ Two Player Mode
✔️ Smart Game State Management
✔️ Win Detection (Rows, Columns, Diagonals)
✔️ Draw Detection
✔️ Restart Game Button
✔️ Visual Highlight for Winner
✔️ Smooth UI Experience

```bash
git clone https://github.com/eslammasry1/tic-tac-toe.git
```

Open in **Visual Studio** and run the project.

---

## 🎮 Gameplay

| Player   | Symbol |
| -------- | ------ |
| Player 1 | ❌ X    |
| Player 2 | ⭕ O    |

* Players take turns
* Click any empty cell to play
* First to align 3 symbols wins
* If all cells are filled → It's a Draw
